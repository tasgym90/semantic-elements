Lesson 01 Homework Assignment - Summary of Semantic Elements

The purpose of semantic elements is to provide helpful information for both web developers and browsers. Semantic elements help to define the content that exists on a web page by giving browsers directions for how the information should be displayed. Depending on their purpose and characteristics, semantic elements will manifest differently on a webpage. For example, they help to distinguish the header from the body of a paragraph.
